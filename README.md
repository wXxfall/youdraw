# 你画我猜 - 双人在线游戏

一个基于WebSocket的双人在线绘画猜词游戏。

## 功能特性

- ✅ 双人实时在线游戏
- ✅ 自动角色分配（画画者/猜题者）
- ✅ 实时画板同步
- ✅ 多种颜色和画笔粗细
- ✅ 橡皮擦功能
- ✅ 答案校验系统
- ✅ 双方同意进入下一关
- ✅ 身份互换机制
- ✅ 分数统计
- ✅ 响应式设计，支持移动端

## 技术栈

- **后端**: Node.js + Express + Socket.io
- **前端**: HTML5 + CSS3 + JavaScript + Canvas API
- **实时通信**: WebSocket (Socket.io)

## 安装和运行

### 1. 安装依赖

```bash
cd youdraw
npm install
```

### 2. 启动服务器

```bash
npm start
```

或使用开发模式（自动重启）：

```bash
npm run dev
```

### 3. 访问游戏

打开浏览器访问: `http://localhost:3001`

## 游戏玩法

1. **加入游戏**
   - 输入你的昵称
   - 输入房间号（和朋友约定相同的房间号）
   - 点击"加入游戏"

2. **等待匹配**
   - 第一个进入房间的玩家会看到等待界面
   - 第二个玩家加入后游戏自动开始

3. **游戏流程**
   - **画画者**: 看到题目，在画板上绘画
   - **猜题者**: 看画板，输入答案猜测
   - 答对后显示庆祝动画
   - 双方都同意后进入下一关
   - 每回合身份互换

4. **画板工具**
   - 8种颜色可选
   - 3种画笔粗细（细/中/粗）
   - 铅笔工具
   - 橡皮擦工具
   - 清空画板

## 注意事项

1. 确保两个玩家在同一网络下或服务器可访问
2. 房间号由玩家自行约定
3. 每个房间最多2人
4. 玩家离开后房间会自动销毁

## 开发说明

### 添加新词语

编辑 `server.js` 中的 `WORDS` 数组：

```javascript
const WORDS = [
  '你的词语1',
  '你的词语2',
  // ...
];
```

### 自定义端口

修改 `server.js` 中的端口号：

```javascript
const PORT = process.env.PORT || 3001; // 改为你想要的端口
```
